Welcome to GaiaAlertsPy
========================================


.. image:: ../logo/Gapy_logo.png
   :align: center
   :alt: GaiaAlertsPy logo
   :width: 400


Welcome to the GaiaAlertsPy documentation! This package provides tools to work with Gaia photometric science alert data. **GaiaAlertsPy** is a Python library for scraping the the Gaia Photometric Science Alerts.

Do you want the Gaia DR3 ID's for the alerts? Please check out our crossmatch table at Zenodo `Gaia Photometric Science Alerts Crossmatch with Gaia DR3 - Feb. 2024 <https://zenodo.org/records/10823036>`_.

The data processing pipelines, survey description, and performance of the Gaia Photometric Science Alerts can be found in `Hodgkin et al. 2021 <https://ui.adsabs.harvard.edu/abs/2021A%26A...652A..76H/abstract>`_.
For more information please check out the `Gaia Alerts <https://gsaweb.ast.cam.ac.uk/alerts/>`_ page.


Questions, or looking for specific features? Please consider opening an issue on the `GitHub repository <https://github.com/AndyTza/GaiaAlertsPy/issues>`_. 
For more questions or suggestions, please contact the author `Andy Tzanidakis <mailto:atzanida@uw.edu>`_.
