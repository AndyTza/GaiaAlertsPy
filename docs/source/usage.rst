Installation
------------

To use GaiaAlertsPy, first install it using pip:

.. code-block:: console

   pip install GaiaAlertsPy

Alternatively, you can clone and install the most recent version of GaiaAlertsPy:
.. code-block:: console
   git clone https://github.com/AndyTza/GaiaAlertsPy.git
   cd GaiaAlertsPy
   pip install .
